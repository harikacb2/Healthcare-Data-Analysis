# 🏥 Healthcare Data Analysis

## 📌 Project Overview
This project analyzes hospital patient data using Excel, Power Query, and Power BI.

## 🎯 Objectives
- Analyze patient demographics
- Study treatment costs
- Measure readmission rates
- Evaluate patient satisfaction

## 🛠 Tools Used
- Microsoft Excel
- Power Query
- Power BI

## 📂 Dataset
- 984 Patient Records
- 10 Variables
